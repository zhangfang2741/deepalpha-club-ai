# 机构资金信号 · 判断逻辑说明书

> 对应代码：`app/services/institutional_signals/`（constants / fetchers / dimensions / states / calculator）
> 目的：把「数据 → 信号 → 维度分 → 状态」的每一步阈值与公式讲清楚，可复核、可调参。

---

## 0. 总览：一条决策链，五个问题

| 维度 | 核心问题 | 状态 | 数据源 |
|------|----------|------|--------|
| Expectation 预期 | 分析师在重新定价未来吗？ | ✅ 已接入 | FMP |
| Positioning 仓位 | 资金已经在下注了吗？ | ✅ 已接入（快照） | yfinance 期权 |
| Participation 参与度 | 其他资金跟了吗？ | ✅ 已接入 | FMP 日线 |
| Fundamental 基本面 | 经营真的变好了吗？ | ⏳ Phase 3 | FMP 财报/Transcript |
| Confirmation 确认 | 长期资金确认了吗？ | ⏳ Phase 3 | Insider/13F/ETF |

每个维度输出 **0–100 子分**；缺失维度按**中性 50 计入**综合分（不剔除），并计入**覆盖度**与**置信度**。

---

## 1. Expectation 预期

**数据源**：FMP `price-target-summary`（目标价近月/季/年均值 + 家数）、`grades-historical`（各月强买/买/持有/卖出家数）。

**基准分 50**，逐项加减：

| 信号 | 公式 | 阈值 → 加减分 | hit |
|------|------|---------------|-----|
| 目标价修正 | `mom = (近月均值 − 近季均值) / 近季均值` | ≥ +5% → +20；≥ +1% → +12；≤ −1% → −15 | ↑上调时 |
| 评级共识 | `买入占比 = (强买+买入) / 总家数`，比最近两个有效月 | 环比 +2pt → +15；环比 −2pt → −15；占比 ≥ 60% → +8 | 环比上升时 |
| EPS/营收修正 | — | 待每日快照库（Phase 4） | 恒 false |

**门槛**：目标价样本数 `lastMonthCount < 2` 不采信。两项均无有效数据 → `status=partial`。

---

## 2. Positioning 仓位（期权快照）

**数据源**：yfinance `option_chain`，聚合 **45 天内近月合约**（不足 2 个到期日则取最近 2 个）。

> ⚠️ 关键限制：yfinance 是**快照**，只有当下的量/OI/IV。手册里的「OI↑ / IV↑ / IV Rank」是**变化率**，需历史基线。本版用快照能算实的三个量，变化率与 Rank 标注为「待每日快照库」。

聚合量：`call_vol / put_vol / call_oi / put_oi`（求和），`atm_iv`（行权价最接近现价的 Call+Put 的 IV 均值，年化小数）。

**基准分 50**：

| 信号 | 公式 | 阈值 → 加减分 | hit |
|------|------|---------------|-----|
| Call 资金流 | `PCR量 = put_vol/call_vol`；`量/仓 = call_vol/call_oi` | PCR量 ≤ 0.7 **且** 量/仓 ≥ 0.4 → +20；PCR量 ≤ 0.9 → +8 | 强看涨下注时 |
| 看跌压力 | `PCR量` 与 `PCR仓 = put_oi/call_oi` | 任一 ≥ 1.3 → −15 | put 主导时（方向↓） |
| 隐含波动率 | `atm_iv` | ≥ 60% → +6；≥ 40% → +3 | IV 偏高时 |

**读法**：`量/仓比` 高 = 当日新增下注活跃（「有资金提前下注」的快照代理）；PCR 低 = 看涨；IV 高 = 市场预期有事件。

---

## 3. Participation 参与度

**数据源**：FMP `historical-price-eod`（近 60 天日线）。窗口 = 不含当日的**前 20 个交易日**。

**基准分 50**：

| 信号 | 公式 | 阈值 → 加减分 | hit |
|------|------|---------------|-----|
| 相对成交量 | `当日量 / 20日均量` | ≥ 2.0 → +25；≥ 1.5 → +15；≥ 1.2 → +8；< 0.8 → −10 | ≥ 1.5 时 |
| 跳空幅度 | `(当日开 − 昨收)/昨收` | ≥ +2% → +10；≤ −2% → −10 | 幅度达标时 |
| 价格突破 | 当日收盘 vs 20日高/低 | 破 20 日高 → +15；破 20 日低 → −15 | 突破/跌破时 |

历史不足 21 天 → `status=partial`。

---

## 4. 综合分与置信度（本次修正的核心）

**综合分**（权重）：预期 0.30 · 仓位 0.25 · 参与度 0.20 · 基本面 0.15 · 确认 0.10。

```
composite = Σ(维度分 × 权重) / Σ权重
```

**修正点**：缺失维度**不再剔除**，按中性 50 计入 → 半盘信息不会拉出满分结论。
例：预期 82、参与度 73，其余三维缺失（各 50）→ 综合分 ≈ 64（而非旧算法的 78）。

**覆盖度** = status 为 ok/partial 的维度数（满分 5）。
**置信度**：覆盖 ≥ 4 → 高；≥ 2 → 中；否则低。

## 5. 状态引擎（组合 → 标签）

状态由各维度的 **hit 布尔量 + 方向**组合触发，附证据链。命中多个则全列；无命中 → ⚪ 中性。

| 状态 | 触发条件（代码口径） | 星级 |
|------|----------------------|------|
| 📈 市场预期提升 | 目标价↑hit **且** 评级共识↑hit | ★★★★★ |
| 🚀 趋势确认 | 相对成交量↑hit **且** 价格突破↑hit **且** 预期分 ≥ 55 | ★★★★ |
| 🔥 机构建仓 | 预期分 ≥ 55 **且** Call资金流↑hit **且** IV↑hit **且** 相对成交量↑hit | ★★★★★ |
| 💰 真资金进入 | Call资金流↑hit **且** IV↑hit **且** 未突破 | ★★★★★ |
| ⚡ 事件交易 | IV↑hit **且** Call资金流↑hit **且** 预期分 < 55 **且** 未放量 **且** 非机构建仓 | ★★★★ |
| ❄ 资金撤退 | 评级共识↓hit **且**（看跌压力↓hit 或 内部减持↓hit） | ★★★★ |

**与手册的一处偏差（如实标注）**：手册用「OI 基本不变」区分「事件交易 vs 真机构」。快照无 OI 变化率，本版改用**「预期与现货是否跟上」**作代理——预期强+放量 → 机构建仓；预期弱+未放量 → 事件交易。等每日快照库上线后可换回真实 OI 变化率。

## 6. 已知限制 → 下一步基建

1. **每日快照库**（`SignalSnapshot`）：解锁 EPS/营收修正趋势、期权 OI 变化率、IV Rank —— 把多个「水平」信号升级为「变化率」信号。这是让「连续三次上调」「OI 新增」这类手册核心信号成立的前提。
2. Fundamental / Confirmation 两维接入（Phase 3）。
3. 每日全 universe 扫描出「今日机构建仓榜」（Phase 4，讨论用发现型入口）。
